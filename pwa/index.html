About

Zayvora

Sovereign AI interface built by ViaDecide. No external dependencies. No cloud auth. Everything runs locally. Everything is owned.

Zayvora is a reasoning engine designed for deep, auditable, multi-layer thinking. This PWA connects to your local Zayvora inference server.

v1.0.0 · Built sovereign · daxini.xyz

Terms of Sovereignty

1. Direct Ownership

The user maintains absolute ownership of ALL data generated, retrieved, or processed within this interface. Zayvora is a local-first portal; no data is harvested, analyzed, or stored by any central authority.

2. Local Autonomy

By using Zayvora, you agree to manage your own compute environment. All inference occurs on local hardware or user-controlled API endpoints. There is no cloud fallback, and thus no middle-man.

3. Irrevocable Encryption

Your communication history is protected by AES-GCM 256-bit encryption. The decryption key is derived strictly from your local password. If the password is lost, the data is mathematically unrecoverable.

Privacy Policy

Zero Telemetry

We do not track. There are no analytics pings, no diagnostic reports, and no heartbeat signals sent to external servers. Your instance of Zayvora is a dark node on the network.

No Third-Party Scripts

This application uses zero external libraries or CDNs. Every line of code runs from your local shell, preventing cross-site tracking and script injection.

Permanent Local Storage

Logs and preferences reside only in your browser's IndexedDB. We have no access to this data. Privacy is enforced by physical isolation.
